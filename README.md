# RAG Bot Skeleton — универсальный шаблон для вопросно-ответного бота

**Технологии**: LangChain, ChromaDB, Ollama, HuggingFace Embeddings

**Возможности**:
- Загружает любые текстовые файлы из папки `docs`
- Строит векторную базу для семантического поиска
- Отвечает на вопросы, используя локальную LLM (Ollama)
- Легко переключается на другую тему простой заменой файлов

**Как запустить**:
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить Ollama: `ollama serve`
3. Выполнить скрипт: `python skeleton.py`
4. Задать вопрос по содержимому `docs`

**Пример использования**:
> Вопрос: "О чём этот текст?"  
> Ответ: "Текст описывает..."

**Почему это важно**:  
Это не готовый бот, а **инструмент**, который можно адаптировать под любую задачу:  
- FAQ для компании  
- Поддержка клиентов  
- Анализ документов  
- Персональный ассистент
