# 🧠 RAG Bot Skeleton — универсальный шаблон для вопросно-ответного бота

**Автор:** GlebShalunkov
**Дата:** Апрель 2026  
**Технологии:** Python, LangChain, ChromaDB, Ollama, HuggingFace Embeddings

---

## 📌 О проекте

Это **универсальный скелет** RAG-бота (Retrieval-Augmented Generation), который:
- Загружает любые текстовые файлы из папки `docs/`
- Строит векторную базу для семантического поиска
- Отвечает на вопросы на естественном языке, используя локальную LLM (Ollama)

**Главная особенность:** меняешь документы в `docs/` — меняется тематика бота.  
Не нужно переписывать код под каждый новый кейс.

---

## 🎯 Бизнес‑ценность

Какой бизнес‑задачу решает этот проект?

| Проблема | Решение |
|----------|---------|
| Сотрудники тратят часы на поиск информации в документации | Бот отвечает на вопросы за секунды |
| Новым специалистам нужен «живой» помощник, а не статичный FAQ | Бот работает 24/7, отвечает по‑человечески |
| Компании хотят внедрить AI, но не хотят отправлять данные в облачные API | Бот **полностью локальный** — без API‑ключей, без передачи данных третьим лицам |

**Где пригодится:**
- Внутренний помощник для IT‑команды (документация, гайды)
- FAQ‑бот для клиентов (по правилам, инструкциям)
- Персональный ассистент по личным заметкам / учебным материалам

---

## 🛠️ Стек технологий

| Компонент | Что делает |
|-----------|------------|
| **LangChain** | Оркестрация RAG‑пайплайна |
| **ChromaDB** | Векторная база данных (хранит эмбеддинги) |
| **Ollama** | Запуск LLM локально (без интернета) |
| **HuggingFace Embeddings** | Превращает текст в векторы (модель `paraphrase-multilingual-MiniLM-L12-v2`) |
